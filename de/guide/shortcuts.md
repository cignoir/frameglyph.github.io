# Tastenkombinationen

![Shortcuts settings tab](/images/screenshots/shortcuts-dialog.png)

Sämtliche hier verzeichneten Tastatur-Shortcuts lassen sich in Frameglyph jederzeit komplett personalisiert unter **Einstellungen (Preferences) > Kürzel (Shortcuts)** neu mit eigenen Hotkey-Zuweisungen festlegen.

## Wiedergabesteuerung (Playback)

| Zugewiesene Aktion | Vorgegebene Standard-Kürzel |
|---|---|
| Wiedergabe & Pause ein- & ausschalten (Play / Pause) | `Space (Leertaste)` |
| Ein Frame (Bild) chronologisch voran springen (Step Forward) | `Right ➡ (Pfeiltaste Rechts)` |
| Ein Frame (Bild) chronologisch rückwärts springen (Step Backward) | `Left ⬅ (Pfeiltaste Links)` |
| Ganz zurück zum kompletten Projektanfang springen (Jump to Start) | `Home (Pos1)` |
| Ganz an das Projektende der Spur springen (Jump to End) | `End (Ende)` |
| Die Abspiel-Rotations-Rate stufenartig erhöhen (Speed Up) | `Ctrl (Strg) + Up ⬆ (Pfeiltaste Hoch)` |
| Die Abspiel-Rotations-Rate stufenartig drosseln (Slow Down) | `Ctrl (Strg) + Down ⬇ (Pfeiltaste Runter)` |
| Den Start-Bereich für die Endlosschleife am Clip markieren (Set Loop In) | `I` |
| Den End-Bereich für die Endlosschleife am Clip markieren (Set Loop Out) | `O` |
| Die Loop-Schleifen-Funktionalität sofort an- u. abschalten (Toggle Loop) | `Shift (Umschalt) + L` |
| Einen Loop-Schleifen-Einstieg direkt von der Cursor-Nadel aus festlegen (Range Start at Playhead) | `[` |
| Das Loop-Schleifen-Ende direkt von der derzeitigen Cursor-Nadel aus begrenzen (Range End at Playhead) | `]` |

## Audioregelung (Audio)

| Zugewiesene Aktion | Vorgegebene Standard-Kürzel |
|---|---|
| Allgemeine Tonlautstärke des Pegels anheben (Volume Up) | `Up ⬆ (Pfeiltaste Hoch)` |
| Allgemeine Tonlautstärke des Pegels drosseln (Volume Down) | `Down ⬇ (Pfeiltaste Runter)` |
| Sofortiges Komplett-Stummschalten oder Re-Aktivieren der Soundspuren (Toggle Mute) | `M` |

## Zeitleisten-Lesezeichen (Bookmarks)

| Zugewiesene Aktion | Vorgegebene Standard-Kürzel |
|---|---|
| Einen festen neuen Lesezeichen-Marker zum Zeitraster hinzufügen (Add Bookmark) | `B` |
| Zum chronologisch davor liegenden Lesezeichen-Marker links springen (Previous Bookmark) | `Shift (Umschalt) + B` |
| Den Cursor auf die Markierung zum chronologisch danach liegenden rechten Marker verlagern (Next Bookmark) | `Alt + B` |

## Bearbeitungs-Grundlagen (Edit)

| Zugewiesene Aktion | Vorgegebene Standard-Kürzel |
|---|---|
| Sämtliches komplett im Rasterwerk simultan als ausgewählt aktivieren (Select All) | `Ctrl (Strg) + A` |
| Den Fokus beenden u. alle aktuellen Auswahlen / Elemente de-selektieren (Deselect All) | `Esc (Escape-Taste)` |
| Unverzügliches Löschen aller gegenwärtig via Rahmen / Cursor markierten Zielelemente (Delete Selected) | `Del (Entf-Taste)` |

## Datei- & Projektverwaltung (File)

| Zugewiesene Aktion | Vorgegebene Standard-Kürzel |
|---|---|
| Medien-Dateien via Systemfenster als neues Element zu Slots in die Videotimeline hereinladen (Import Video) | `Ctrl (Strg) + I` |
| Schnelles festes lokales Überschreiben/Sichern des jetzigen Projektstandes (Save Project) | `Ctrl (Strg) + S` |
| Aufruf des Dialogfensters, um die Projekte mit neuen Namen/als Backup explizit neu anzulegen (Save Project As) | `Ctrl (Strg) + Shift (Umschalt) + S` |

## Lokales Mitschnittsystem (Capture)

| Zugewiesene Aktion | Vorgegebene Standard-Kürzel |
|---|---|
| Automatisches sofortiges Auslösen u. auch festes Beenden des Bildschirm-Rekorders der App (Start / Stop Capture) | `Ctrl (Strg) + Shift (Umschalt) + 0` |

## Raster & Timeline-Ansicht (Timeline)

| Zugewiesene Aktion | Vorgegebene Standard-Kürzel |
|---|---|
| Erweitern/Schrumpfen der Zeitleiste via Auto-Zoom zur Übersicht all dessen im Monitorbereich (Fit Timeline to View) | `F` |
