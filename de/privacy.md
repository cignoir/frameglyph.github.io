# Datenschutzrichtlinie

*Zuletzt aktualisiert: Februar 2026*

## Überblick

Frameglyph ist eine Desktop-Anwendung, die vollständig auf Ihrem lokalen Computer ausgeführt wird. Wir verpflichten uns, Ihre Privatsphäre zu schützen.

## Datenerfassung

Frameglyph sammelt, überträgt oder speichert **keine** persönlichen Daten. Alle Ihre Projektdateien, Videos und Annotationen verbleiben auf Ihrem lokalen Gerät.

## Netzwerkzugriff

Frameglyph benötigt keine Internetverbindung, um zu funktionieren. Es werden keine Daten an externe Server gesendet.

## Dienste von Drittanbietern

Wenn die Anwendung über Steam vertrieben wird, kann der Steam-Client Nutzungsdaten gemäß der [Datenschutzrichtlinie von Valve](https://store.steampowered.com/privacy_agreement/) sammeln. Dies liegt außerhalb des Geltungsbereichs von Frameglyph selbst.
