# Support et Aide

## Foire Aux Questions (FAQ)

### Quelle est la configuration requise ?
Frameglyph fonctionne sous Windows. FFmpeg est requis pour les fonctionnalités d'exportation vidéo.

### Comment installer FFmpeg ?
Téléchargez FFmpeg depuis [ffmpeg.org](https://ffmpeg.org/download.html) et définissez le chemin dans Préférences > Système > Configuration FFmpeg (Preferences > System > FFmpeg Configuration).

### Mon exportation a échoué. Que dois-je vérifier ?
1. Vérifiez que FFmpeg est correctement configuré dans les Préférences.
2. Assurez-vous que le répertoire de destination existe et dispose des droits d'écriture.
3. Vérifiez que le fichier vidéo source n'est pas corrompu.

### Puis-je utiliser Frameglyph sans FFmpeg ?
Vous pouvez réviser et annoter des vidéos sans FFmpeg, mais les fonctionnalités d'exportation le nécessitent.
