# はじめに

![Frameglyphの概要](/images/screenshots/hero.png)

Frameglyphは、Windows向けの動画レビュー＆アノテーションアプリケーションです。フレーム単位の正確な再生、直感的な描画ツール、強力なエクスポート機能を備え、アニメーター、ゲーム開発者、映像レビュアーのシームレスなフィードバックを支援するために設計されています。

## 主な機能

- **動画再生** — フレーム単位のナビゲーション、変速再生、オーディオスクラブ、ループ制御
- **アノテーション** — フリーハンド描画（Live Ink）、筆圧感知対応、図形、テキストオーバーレイ
- **タイムライン** — ブックマーク、範囲選択、キーフレーム・トラック管理
- **エクスポート** — MP4、WebM、GIF、APNG、連番画像（アノテーション焼き付け）
- **マルチ動画比較** — 最大4本の動画をレイアウト（1x1、2x1、2x2）で同時表示
- **スクリーンキャプチャ** — 内蔵録画からダイレクトにレビューへインポート
- **プロジェクト管理** — レビューセッションと各種設定の保存・読み込み
