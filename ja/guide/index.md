# はじめに

![Frameglyphの概要](/images/screenshots/hero.png)

Frameglyphは、Windows向けの動画レビュー＆アノテーションアプリケーションです。フレーム単位の正確な再生、描画ツール、エクスポート機能を備え、アニメーター、ゲーム開発者、映像レビュアーのために設計されています。

## 主な機能

- **動画再生** — フレーム単位のナビゲーション、変速再生、ループ制御
- **アノテーション** — フリーハンド描画（Live Ink）、図形、テキストオーバーレイ
- **タイムライン** — ブックマーク、範囲選択、キーフレーム管理
- **エクスポート** — MP4、WebM、GIF、PNG/JPEG/WebPシーケンス（アノテーション焼き付け）
- **マルチ動画比較** — 最大9本の動画をレイアウト（1x1、2x1、2x2、3x3）で同時表示
- **スクリーンキャプチャ** — 内蔵録画とダイレクトインポート
- **プロジェクト管理** — レビューセッションの保存と読み込み
