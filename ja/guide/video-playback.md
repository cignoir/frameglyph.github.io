# 動画再生

Frameglyphは、フレームレベルの精度で動画のインポートとレビューをサポートします。

## 動画のインポート

- 動画ファイルをアプリケーションウィンドウにドラッグ＆ドロップします。
- または、メニューバーの **ファイル > 動画をインポート** を使用します。

対応フォーマット: MP4、WebMなど、ブラウザのビデオエンジンがサポートするフォーマット。

## 再生コントロール

| 操作 | 説明 |
|---|---|
| 再生 / 一時停止 | 再生の開始または停止 |
| コマ送り | 1フレーム進む |
| コマ戻し | 1フレーム戻る |
| 先頭へジャンプ | 最初のフレームへ移動 |
| 末尾へジャンプ | 最後のフレームへ移動 |
| 速度アップ | 再生速度を上げる |
| 速度ダウン | 再生速度を下げる |

## ループ再生

1. 開始フレームで **ループイン** ポイントを設定します。
2. 終了フレームで **ループアウト** ポイントを設定します。
3. ループモードを切り替えて、選択範囲をリピート再生します。

## マルチビデオレイアウト

Frameglyphは最大4本の動画を同時に表示でき、サイドバイサイド比較が可能なレイアウトに対応しています。

全再生コントロールの一覧は[キーボードショートカット](./shortcuts)をご覧ください。
