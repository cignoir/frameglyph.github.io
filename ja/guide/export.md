# エクスポート

FrameglyphはFFmpegを使用して、アノテーションを焼き付けた動画をエクスポートします。

## 前提条件

エクスポートにはFFmpegの設定が必要です。[インストール](./installation#ffmpegのセットアップ)を参照してください。

![エクスポート設定ダイアログ](/images/screenshots/export-dialog.png)

## エクスポートフォーマット

| フォーマット | 説明 |
|---|---|
| **MP4 (H.264)** | 標準的な動画フォーマット、広い互換性 |
| **WebM (VP9)** | オープンフォーマット、良好な圧縮率 |
| **GIF アニメーション** | アニメーションGIF、音声非対応 |
| **APNG アニメーション** | アニメーションPNG、GIFより高品質、音声非対応 |
| **PNG 連番** | ロスレス画像シーケンス |
| **JPEG 連番** | 圧縮画像シーケンス |
| **WebP 連番** | モダンな画像シーケンスフォーマット |

## エクスポート設定

### 解像度
**オリジナル**、**1080p**、**720p**、**480p**から選択できます。

### フレームレート (FPS)
標準的なフレームレート（24、30、60など）から選択するか、**自動（最大入力 FPS に合わせる）** を使用してソース動画のフレームレートを引き継ぎます。

### 品質
- **RAW**（最高品質 / 大容量）
- **高**（高品質 / バランス）
- **中**（標準 / 小容量）
- **低**（ドラフト / 最小容量）

### 範囲
- **全体**: クリップ全体をエクスポート
- **選択範囲のみ**: 選択範囲（イン/アウトポイント）のみをエクスポート

### オーバーレイ（焼き込み）
**キャプション、アノテーション、Live Ink** を出力動画に焼き付けるかどうかを切り替えます。

## GIF と APNG のヒント

::: warning
GIF および APNG ファイルは非常に大きくなる可能性があります。最良の結果を得るには：
- 低解像度（480p）を使用
- 低フレームレート（10-15 fps）を使用
- クリップを短くする
- これらのフォーマットでは音声はサポートされていません
:::
