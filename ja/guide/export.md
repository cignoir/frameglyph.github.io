# エクスポート

FrameglyphはFFmpegを使用して、アノテーションを焼き付けた動画をエクスポートします。

## 前提条件

エクスポートにはFFmpegの設定が必要です。[インストール](./installation#ffmpegのセットアップ)を参照してください。

## エクスポートフォーマット

| フォーマット | 説明 |
|---|---|
| **MP4 (H.264)** | 標準的な動画フォーマット、広い互換性 |
| **WebM (VP9)** | オープンフォーマット、良好な圧縮率 |
| **GIF** | アニメーションGIF、音声非対応 |
| **PNGシーケンス** | ロスレス画像シーケンス |
| **JPEGシーケンス** | 圧縮画像シーケンス |
| **WebPシーケンス** | モダンな画像シーケンスフォーマット |

## エクスポート設定

### 解像度
オリジナル、1080p、720p、480pから選択できます。

### フレームレート
標準的なフレームレート（24fps、30fps、60fpsなど）から選択するか、自動でソース動画に合わせます。

### 品質
- **Raw**: 最高品質、最大ファイルサイズ
- **高**: 良好な品質、バランスの取れたファイルサイズ
- **中**: 標準品質、小さめのファイルサイズ
- **低**: ドラフト品質、最小ファイルサイズ

### 範囲
- **全体**: 動画全体をエクスポート
- **選択範囲のみ**: イン/アウト範囲のみエクスポート

### オーバーレイ
アノテーションとキャプションを出力に焼き付けるかどうかを切り替えます。

## GIFのヒント

::: warning
GIFファイルは非常に大きくなることがあります。最良の結果を得るには:
- 低解像度（480p）を使用
- 低フレームレート（10-15 fps）を使用
- クリップを短くする
- GIFフォーマットでは音声はサポートされません
:::
