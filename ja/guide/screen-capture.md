# スクリーンキャプチャ

Frameglyphには画面録画機能が内蔵されており、録画した動画をそのままレビューワークフローへ取り込むことができます。

## 録画

1. **キャプチャ開始 / 停止** ショートカットまたはツールバーボタンで録画を開始します。
2. 録画する領域、アプリケーション、または特定のウィンドウを選択します。
3. 録画が完了したら停止します。

キャプチャした動画はFrameglyphの新しいスロットへ自動的にインポートされます。

## 設定

**設定 > キャプチャ** でスクリーンキャプチャを構成できます：

| 設定 | 説明 |
|---|---|
| **保存場所** | 録画が保存されるディレクトリ |
| **フレームレート (FPS)** | 録画FPS（高いほど滑らかだがファイルが大きくなる） |
| **フォーマット** | WebM（ネイティブ）、MP4、APNG、GIF（MP4/APNG/GIFにはFFmpegが必要） |
| **マウスカーソルを非表示** | スクリーンキャプチャからマウスカーソルを除外する |
| **クリップボードにコピー** | 保存後に録画ファイルを自動的にコピーする |
| **ウィンドウを表示** | 録画終了後にメインウィンドウを表示する（トレイに最小化時） |

::: tip
MP4、APNG、またはGIFへの変換に失敗した場合、WebMがフォールバックとして保存されます。
:::
