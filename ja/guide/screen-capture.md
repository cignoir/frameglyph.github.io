# スクリーンキャプチャ

Frameglyphには画面録画機能が内蔵されており、録画した動画をそのままレビューワークフローに取り込めます。

## 録画

1. キャプチャショートカットまたはツールバーボタンでキャプチャを開始します。
2. 録画する領域またはウィンドウを選択します。
3. 録画が完了したら停止します。

キャプチャした動画はFrameglyphに自動的にインポートされます。

## 設定

**設定 > キャプチャ** でスクリーンキャプチャを構成できます:

| 設定 | 説明 |
|---|---|
| **保存先** | 録画が保存されるディレクトリ |
| **フレームレート** | 録画FPS（高い＝滑らかだがファイルが大きくなる） |
| **フォーマット** | WebM（ネイティブ）、MP4、GIF（MP4/GIFにはFFmpegが必要） |
| **クリップボードにコピー** | 録画後にファイルを自動コピー |
| **ウィンドウを表示** | 録画終了後にメインウィンドウを表示 |

::: tip
MP4またはGIFへの変換に失敗した場合、WebMがフォールバックとして保存されます。
:::
