# キーボードショートカット

![ショートカット設定タブ](/images/screenshots/shortcuts-dialog.png)

すべてのショートカットは **設定 > ショートカット** でカスタマイズできます。

## 再生

| 操作 | デフォルトショートカット |
|---|---|
| 再生 / 一時停止 | `Space` |
| 1フレーム進む | `Right` |
| 1フレーム戻る | `Left` |
| 先頭へジャンプ | `Home` |
| 末尾へジャンプ | `End` |
| 速度を上げる | `Ctrl + Up` |
| 速度を下げる | `Ctrl + Down` |
| ループ開始点を設定 | `I` |
| ループ終了点を設定 | `O` |
| ループ切替 | `Shift + L` |
| 範囲開始を再生位置に | `[` |
| 範囲終了を再生位置に | `]` |

## オーディオ

| 操作 | デフォルトショートカット |
|---|---|
| 音量を上げる | `Up` |
| 音量を下げる | `Down` |
| ミュート切替 | `M` |

## ブックマーク

| 操作 | デフォルトショートカット |
|---|---|
| ブックマーク追加 | `B` |
| 前のブックマーク | `Shift + B` |
| 次のブックマーク | `Alt + B` |

## 編集

| 操作 | デフォルトショートカット |
|---|---|
| すべて選択 | `Ctrl + A` |
| 選択解除 | `Esc` |
| 選択を削除 | `Del` |

## ファイル

| 操作 | デフォルトショートカット |
|---|---|
| 動画をインポート | `Ctrl + I` |
| プロジェクトを保存 | `Ctrl + S` |
| 名前を付けて保存 | `Ctrl + Shift + S` |

## キャプチャ

| 操作 | デフォルトショートカット |
|---|---|
| キャプチャ開始 / 停止 | `Ctrl + Shift + 0` |

## タイムライン

| 操作 | デフォルトショートカット |
|---|---|
| タイムラインを全体表示 | `F` |
