# キーボードショートカット

すべてのショートカットは **設定 > ショートカット** でカスタマイズできます。

## 再生

| 操作 | デフォルトショートカット |
|---|---|
| 再生 / 一時停止 | `Space` |
| コマ送り | `Right` |
| コマ戻し | `Left` |
| 先頭へジャンプ | `Home` |
| 末尾へジャンプ | `End` |
| 速度アップ | `Ctrl + Up` |
| 速度ダウン | `Ctrl + Down` |
| ループイン設定 | `I` |
| ループアウト設定 | `O` |
| ループ切替 | `Shift + L` |

## オーディオ

| 操作 | デフォルトショートカット |
|---|---|
| 音量アップ | `Up` |
| 音量ダウン | `Down` |
| ミュート切替 | `M` |

## ブックマーク

| 操作 | デフォルトショートカット |
|---|---|
| ブックマーク追加 | `B` |
| 前のブックマーク | `Shift + B` |
| 次のブックマーク | `Alt + B` |

## 編集

| 操作 | デフォルトショートカット |
|---|---|
| すべて選択 | `Ctrl + A` |
| 選択解除 | `Esc` |
| 選択を削除 | `Del` |

## ファイル

| 操作 | デフォルトショートカット |
|---|---|
| 動画をインポート | `Ctrl + I` |
| プロジェクトを保存 | `Ctrl + S` |
| 名前を付けて保存 | `Ctrl + Shift + S` |

## キャプチャ

| 操作 | デフォルトショートカット |
|---|---|
| 録画開始 / 停止 | `Ctrl + Shift + 0` |
