# 简介

![Frameglyph overview](/images/screenshots/hero.png)

Frameglyph 是一款专为 Windows 打造的高精度视频审查与批注应用程序。凭借逐帧精准的播放、表现力丰富的绘图工具以及强大的导出功能——旨在为动画师、游戏开发者和视频审查人员简化反馈工作流程。

## 核心功能

- **视频播放** — 逐帧导航、变速播放、音频拖拽预览 (audio scrubbing)、循环控制
- **批注工具** — 自由绘制（动态墨迹 Live Ink）、压感支持、形状、文本叠加层
- **时间轴** — 书签、范围选择、关键帧和轨道管理
- **导出** — 直接将带有批注的视频渲染导出为 MP4、WebM、GIF、APNG 以及图像序列
- **多视频对比** — 在可自由配置的布局（1x1、2x1、1x2、2x2）中最多同时查看4个视频
- **屏幕录制** — 内置录屏功能并可直接导入您的审查会话中
- **项目管理** — 保存并加载您的审查项目

## 开始使用

1. [安装 Frameglyph](./installation)
2. 导入视频并开始视频审查
3. 添加精确的批注和动态墨迹
4. 导出带批注的视频结果

继续阅读[安装指南](./installation)以设置 Frameglyph。
