# 快捷键

![Shortcuts settings tab](/images/screenshots/shortcuts-dialog.png)

所有快捷键均可在 **首选项 (Preferences) > 快捷键 (Shortcuts)** 中自定义。

## 播放 (Playback)

| 操作 | 默认快捷键 |
|---|---|
| 播放 / 暂停 (Play / Pause) | `Space` |
| 步进 (Step Forward) | `Right ➡` |
| 步退 (Step Backward) | `Left ⬅` |
| 跳转至开头 (Jump to Start) | `Home` |
| 跳转至结尾 (Jump to End) | `End` |
| 加速 (Speed Up) | `Ctrl + Up ⬆` |
| 减速 (Slow Down) | `Ctrl + Down ⬇` |
| 设置循环入点 (Set Loop In) | `I` |
| 设置循环出点 (Set Loop Out) | `O` |
| 切换循环播放 (Toggle Loop) | `Shift + L` |
| 范围起始于播放头 (Range Start at Playhead) | `[` |
| 范围结束于播放头 (Range End at Playhead) | `]` |

## 音频 (Audio)

| 操作 | 默认快捷键 |
|---|---|
| 调高音量 (Volume Up) | `Up ⬆` |
| 调低音量 (Volume Down) | `Down ⬇` |
| 切换静音 (Toggle Mute) | `M` |

## 书签 (Bookmarks)

| 操作 | 默认快捷键 |
|---|---|
| 添加书签 (Add Bookmark) | `B` |
| 上一个书签 (Previous Bookmark) | `Shift + B` |
| 下一个书签 (Next Bookmark) | `Alt + B` |

## 编辑 (Edit)

| 操作 | 默认快捷键 |
|---|---|
| 全选 (Select All) | `Ctrl + A` |
| 取消全选 (Deselect All) | `Esc` |
| 删除选定项 (Delete Selected) | `Del` |

## 文件 (File)

| 操作 | 默认快捷键 |
|---|---|
| 导入视频 (Import Video) | `Ctrl + I` |
| 保存项目 (Save Project) | `Ctrl + S` |
| 另存项目为... (Save Project As) | `Ctrl + Shift + S` |

## 捕捉 (Capture)

| 操作 | 默认快捷键 |
|---|---|
| 开始 / 停止捕捉 (Start / Stop Capture) | `Ctrl + Shift + 0` |

## 时间轴 (Timeline)

| 操作 | 默认快捷键 |
|---|---|
| 适配时间轴至视图 (Fit Timeline to View) | `F` |
