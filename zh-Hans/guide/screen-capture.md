# 屏幕录制

Frameglyph 包含了内置的屏幕录制功能，可直接将您的屏幕操作捕捉并导入到审查工作流中。

## 录制操作

1. 使用 **开始 / 停止捕捉 (Start / Stop Capture)** 快捷键或工具栏按钮开始捕捉。
2. 选择要录制的区域、应用程序或特定窗口。
3. 完成后停止录制。

捕获的视频会自动导入到 Frameglyph 的新槽位中。

## 设置选项

在 **首选项 (Preferences) > 捕获 (Captures)** 中配置屏幕录制功能：

| 设置 | 说明 |
|---|---|
| **保存位置 (Save Location)** | 录制文件保存的目录 |
| **帧率 (Frame Rate FPS)** | 录制帧率（数值较高生出的视频越流畅，但文件越大） |
| **格式 (Format)** | WebM（原生）、MP4、APNG 或 GIF（后三者需依靠 FFmpeg） |
| **隐藏鼠标指针 (Hide Mouse Cursor)** | 在录屏中不显示鼠标光标 |
| **复制到剪贴板 (Copy to Clipboard)** | 保存后将录制的视频文件复制到剪贴板 |
| **显示窗口 (Show Window)** | 录制结束后显示主窗口（前提是最小化到托盘状态下录制） |

::: tip 提示
如果转换为 MP4、APNG 或 GIF 失败，系统将后备保存为 WebM 格式。
:::
