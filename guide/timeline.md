# Timeline

The timeline provides an overview of the video and tools for navigation, bookmarking, and range selection.

## Bookmarks

- Add bookmarks to mark important moments in the video.
- Navigate between bookmarks using keyboard shortcuts or clicking on the timeline.
- Bookmarks are saved with the project file.

## Range Selection

Set In/Out points to define a range on the timeline:
- **Loop In**: Start of the selected range
- **Loop Out**: End of the selected range

The selected range is used for:
- Loop playback
- Exporting a portion of the video

## Keyframe Management

Annotations with keyframes are displayed on the timeline. You can:
- Add keyframes at the current frame
- Move keyframes by dragging
- Delete keyframes from the timeline
