# 화면 캡처

Frameglyph에는 화면 기록 기능을 내장하여 디스플레이되는 작업 화면을 자체 리뷰용 워크플로우로 바로 녹화할 수 있습니다.

## 녹화 방법

1. 실행창의 버튼 혹은 시작 단축키 **Start / Stop Capture**를 클릭해 캡처 작업을 시작합니다.
2. 특정 공간이나 앱 및 전체 윈도우 중 원하는 부분을 선택합니다.
3. 마치려면 녹화를 종료합니다.

캡처된 비디오는 Frameglyph의 다음 슬롯으로 자동 넘어가 작업할 수 있게 됩니다.

## 캡처 세팅

**기본 설정(Preferences) > 화면 캡처(Captures)** 메뉴 내의 구성 조작:

| 설정 항목 | 설명 |
|---|---|
| **저장 위치 (Save Location)** | 캡처 파일 저장 장소 지정 |
| **화면 FPS (Frame Rate)** | 프레임 기록 수치 제어 (숫자가 클 수록 비디오는 부드럽지만 용량 커짐) |
| **파일 포맷 (Format)** | WebM이 기본 세팅, 파일 옵션 중 MP4/APNG/GIF를 거칠경우 FFmpeg 컨픽 필요 |
| **마우스 커서 지우기** | 화살표 등 커서 흔적을 지우기 |
| **클립보드 저장(Copy to Clipboard)** | 레코드가 끝난 뒤 비디오 데이터를 타 서비스에 곧바로 붙이기용 클립보드로 제공 여부 |
| **종료 뒤 화면표시 (Show Window)** | 영상 녹화 후 백그라운드나 트레이 항목 안에 최소화돼 있던 프로그램 창을 메인으로 호출 설정 여부 |

::: tip 안내
GIF, APNG 혹은 MP4 설정의 변환 도중 실패할 경우에는 이전에 만들어진 원본인 WebM 포맷 데이터가 안전하게 대체 및 보호되어 남게 됩니다.
:::
