# 단축키 설정

![Shortcuts settings tab](/images/screenshots/shortcuts-dialog.png)

단축키는 **기본 설정(Preferences) > 단축키(Shortcuts)** 에서 사용자가 정의할 수 있습니다.

## 재생(Playback)

| 작업 | 단축키 기본 설정 |
|---|---|
| 재생/정지 | `Space` |
| 다음 프레임 | `Right` (우방향키) |
| 뒤로 이동 | `Left` (좌방향키) |
| 영상 맨 앞으로 가기 | `Home` |
| 영상 맨 뒤로 가기 | `End` |
| 영상 배속 빨리하기 | `Ctrl + Up` (상방향키) |
| 재생 느리게 지연하기 | `Ctrl + Down` (하방향키) |
| 루프 (IN) 지정 | `I` |
| 루프 (OUT) 지정 | `O` |
| 루프 적용 끄고/켜기 | `Shift + L` |
| 재생막대(Playhead)에서 루프 시작 | `[` |
| 재생막대(Playhead)에서 루프 끝 | `]` |

## 음향 효과(Audio)

| 작업 | 단축키 기본 설정 |
|---|---|
| 볼륨 업 | `Up` |
| 볼륨 다운 | `Down` |
| 음소거(Mute) 작동/해제 | `M` |

## 책갈피 (Bookmarks)

| 작업 | 단축키 기본 설정 |
|---|---|
| 북마크 추가 | `B` |
| 이전 항목 넘김 | `Shift + B` |
| 다음 항목 넘김 | `Alt + B` |

## 편집 모드

| 작업 | 단축키 기본 설정 |
|---|---|
| 모든 항목 골라잡기 (선택) | `Ctrl + A` |
| 선택 내리기 | `Esc` |
| 선택 항목 지우기 | `Del` |

## 파일 컨트롤

| 작업 | 단축키 기본 설정 |
|---|---|
| 미디어 Import 시키기 | `Ctrl + I` |
| 지금 프로젝트 저장 | `Ctrl + S` |
| 새이름으로 지정하여 저장 | `Ctrl + Shift + S` |

## 화면 캡쳐

| 작업 | 단축키 기본 설정 |
|---|---|
| Capture(녹화) 누르고 멈추기 | `Ctrl + Shift + 0` |

## 타임라인

| 작업 | 단축키 기본 설정 |
|---|---|
| 보여지는 화면 기준 영역 사이즈 조절 | `F` |
