# Atajos de Teclado

![Shortcuts settings tab](/images/screenshots/shortcuts-dialog.png)

Todos los atajos se pueden personalizar en **Preferencias (Preferences) > Atajos (Shortcuts)**.

## Reproducción (Playback)

| Acción | Atajo Predeterminado |
|---|---|
| Reproducir / Pausar (Play / Pause) | `Space` |
| Avanzar un fotograma (Step Forward) | `Right ➡` |
| Retroceder un fotograma (Step Backward) | `Left ⬅` |
| Saltar al Inicio (Jump to Start) | `Home` |
| Saltar al Final (Jump to End) | `End` |
| Acelerar (Speed Up) | `Ctrl + Up ⬆` |
| Ralentizar (Slow Down) | `Ctrl + Down ⬇` |
| Establecer Entrada de Bucle (Set Loop In) | `I` |
| Establecer Salida de Bucle (Set Loop Out) | `O` |
| Alternar Bucle (Toggle Loop) | `Shift + L` |
| Rango Empieza en Cabezal (Range Start at Playhead) | `[` |
| Rango Termina en Cabezal (Range End at Playhead) | `]` |

## Audio

| Acción | Atajo Predeterminado |
|---|---|
| Subir Volumen (Volume Up) | `Up ⬆` |
| Bajar Volumen (Volume Down) | `Down ⬇` |
| Alternar Silencio (Toggle Mute) | `M` |

## Marcadores (Bookmarks)

| Acción | Atajo Predeterminado |
|---|---|
| Agregar Marcador (Add Bookmark) | `B` |
| Marcador Anterior (Previous Bookmark) | `Shift + B` |
| Marcador Siguiente (Next Bookmark) | `Alt + B` |

## Edición (Edit)

| Acción | Atajo Predeterminado |
|---|---|
| Seleccionar Todo (Select All) | `Ctrl + A` |
| Deseleccionar Todo (Deselect All) | `Esc` |
| Eliminar Seleccionado (Delete Selected) | `Del` |

## Archivo (File)

| Acción | Atajo Predeterminado |
|---|---|
| Importar Video (Import Video) | `Ctrl + I` |
| Guardar Proyecto (Save Project) | `Ctrl + S` |
| Guardar Proyecto Como (Save Project As) | `Ctrl + Shift + S` |

## Capturar (Capture)

| Acción | Atajo Predeterminado |
|---|---|
| Iniciar / Detener Captura (Start / Stop Capture) | `Ctrl + Shift + 0` |

## Línea de Tiempo (Timeline)

| Acción | Atajo Predeterminado |
|---|---|
| Ajustar Línea de Tiempo a la Vista (Fit Timeline to View) | `F` |
